# Ghost Hunter

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad Ghost Hunter para OpenClaw.

## Qué Soluciona
Detecta vulnerabilidades y amenazas ocultas en el código y la infraestructura.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales y listos para producción
- Enfoque en seguridad primero
- Soporte de reversión

## Uso
/ghost-hunter

## Ejemplo
```
Analiza el código en busca de problemas de seguridad.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
